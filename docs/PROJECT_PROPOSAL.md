# Mini SOC — App Monitoramento de Segurança

Projeto colaborativo para ingestão, análise e visualização de logs de segurança.

## Objetivo
Criar um MVP de SOC com:
- ingestão de logs JSON
- regras simples de detecção
- dashboard web
- exportação de incidentes

## Público
Projeto de portfólio voltado para:
- Cybersecurity
- Desenvolvimento
- Dados
